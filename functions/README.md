# Cloudflare Pages Functions (generated)

**`functions/_middleware.js`** is **created by `npm run build`** from `dist/` via `scripts/sync-seo-functions.mjs`. It runs in front of static assets and serves `/robots.txt`, `/sitemap-*.xml`, and `/og-image.png` with correct content types (Cloudflare Pages can otherwise return the SPA HTML shell for those URLs).

Do not edit `_middleware.js` by hand — change `public/robots.txt`, pages, or OG image, then rebuild.
