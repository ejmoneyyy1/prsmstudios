# Cloudflare Pages Functions (generated)

The `*.js` files in this folder are **created by `npm run build`** from `dist/` via `scripts/sync-seo-functions.mjs`. They ensure `/robots.txt`, `/sitemap-*.xml`, and `/og-image.png` return correct responses on Cloudflare Pages (root static files can otherwise be replaced by the SPA shell).

Do not edit the generated `*.js` files by hand — change `public/robots.txt`, routes, or OG image, then rebuild.
